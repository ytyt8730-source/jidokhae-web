# 아키텍처 결정 기록 (ADR)

## ADR이란?

아키텍처 결정 기록(Architecture Decision Record)은 프로젝트에서 내린 중요한 기술적 결정을 문서화하는 방식입니다.

## 왜 필요한가?

1. **재작업 방지**: 미리 결정된 사항을 명확히 하여 개발 중 충돌 예방
2. **컨텍스트 보존**: 왜 이런 결정을 했는지 나중에 참조 가능
3. **AI 에이전트 가이드**: AI가 일관된 방식으로 코드 생성

## ADR 템플릿

```markdown
# ADR-XXX: [제목]

## 상태
[ ] 제안 | [x] 결정됨 | [ ] 폐기됨

## 날짜
YYYY-MM-DD

## 맥락
- 이 결정이 필요한 배경
- 현재 상황과 문제점

## 고려한 대안
1. 대안 A: 설명 + 장단점
2. 대안 B: 설명 + 장단점

## 결정
- 최종 결정 내용
- 선택 이유

## 결과
- 이 결정으로 인한 영향
- 주의사항
```
