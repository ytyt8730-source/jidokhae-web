## 1. 개선 방향 요약

### 현재 문제점

| 영역 | 현재 상태 | 문제점 |
|------|----------|--------|
| **테마 적용** | CSS 변수 일부 적용 | Electric/Warm 테마 완전 분리 미흡 |
| **헤드라인 폰트** | 고정값 사용 | 테마별 폰트 분기 없음 (Electric=고딕, Warm=명조) |
| **레이아웃** | 전통적 리스트 | Bento Grid + Bottom Sheet 패턴 미적용 |
| **가격 표시** | 텍스트 혼용 | KongIcon + "콩" 단위 일관성 부족 |
| **애니메이션** | 기본 transition | Framer Motion stagger/spring 미적용 |
| **Atmospheric Cover** | 없음 | 책 표지 + 블러 배경 효과 미적용 |

### 개선 후 기대 효과

```
Electric Mode:
  - Cobalt Blue (#0047FF) + Acid Lime (#CCFF00)
  - Sans-serif 헤드라인 (Outfit/Noto Sans KR)
  - 깔끔하고 현대적인 토스/당근마켓 느낌

Warm Mode:
  - Deep Navy (#0F172A) + Burnt Orange (#EA580C)
  - Serif 헤드라인 (Noto Serif KR)
  - Paper texture noise + Kinfolk/Aesop 느낌
```

## 7. 애니메이션 명세

### 7.1 Framer Motion Variants

```tsx
// lib/animations.ts

// 페이지 진입 stagger
export const pageVariants = {
  initial: { opacity: 0 },
  animate: {
    opacity: 1,
    transition: {
      staggerChildren: 0.1,
    },
  },
}

// 카드 진입
export const cardVariants = {
  initial: { opacity: 0, y: 20 },
  animate: {
    opacity: 1,
    y: 0,
    transition: {
      type: 'spring',
      stiffness: 300,
      damping: 30,
    },
  },
}

// 호버
export const hoverVariants = {
  rest: { scale: 1 },
  hover: {
    scale: 1.02,
    transition: { type: 'spring', stiffness: 400, damping: 25 },
  },
}

// Bottom Sheet
export const sheetVariants = {
  hidden: { y: '100%' },
  visible: {
    y: 0,
    transition: {
      type: 'spring',
      stiffness: 300,
      damping: 30,
    },
  },
  exit: {
    y: '100%',
    transition: { duration: 0.2 },
  },
}
```

### 7.2 애니메이션 적용 위치

| 위치 | 애니메이션 | 설정 |
|------|-----------|------|
| 홈 Bento Grid | Stagger reveal | delay: 0.1s between items |
| Session Cards | Stagger + hover | scale 1.02 on hover |
| Bottom Sheet | Spring slide up | stiffness: 300, damping: 30 |
| FAB | Pulse on new notification | subtle glow animation |
| Badge earned | Confetti burst | canvas-confetti |
| MY GROWTH progress | Count up | duration: 1s |
