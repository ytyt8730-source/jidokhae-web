## 1. 테스트 환경 준비

### 1.1 서버 시작

```bash
cd C:\Cursor\jidokhae-web\jidokhae
npm run dev
```

**확인 사항:**
- [ ] 서버 정상 시작 (http://localhost:3000)
- [ ] 컴파일 오류 없음
- [ ] 환경 변수 로드 (.env.local)

### 1.2 필수 환경 변수 확인

```bash
# .env.local 파일에 다음 변수들이 설정되어 있어야 함
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
PORTONE_API_KEY=
PORTONE_API_SECRET=
PORTONE_STORE_ID=
SOLAPI_API_KEY=
SOLAPI_API_SECRET=
SOLAPI_SENDER=
```

### 1.3 데이터베이스 상태 확인

Supabase 대시보드에서 확인:
- [ ] users 테이블 존재
- [ ] meetings 테이블 존재
- [ ] registrations 테이블 존재
- [ ] 기타 필수 테이블 (badges, praises, reviews 등)
