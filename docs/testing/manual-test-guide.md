## 3️⃣ UI/UX 테스트 체크리스트

### 🏠 홈 화면 (`/`)

1. **배너 슬라이드**
   - [됨 ] 배너가 표시되는가?
   - [됨 ] 자동으로 슬라이드되는가? (5초 간격)
   - [됨 ] 좌/우 화살표 클릭 시 전환되는가?

2. **모임 카드**
   - [됨 ] 모임 카드가 순차적으로 나타나는가? (Stagger 애니메이션)
   - [됨 ] 카드에 마우스 올리면 살짝 떠오르는가? (hover: -4px)
   - [되는것 같음 ] 모임 상태 뱃지 표시 (모집중/마감임박/마감)
   - [됨 ] "O명 참여" 형식으로 인원 표시

3. **색상 및 폰트**
   - [ ] 브랜드 색상 (테라코타색 #c77654) 확인 : 브랜드 색상은 이게 아님, 어디서 이 정보가 있었는지 확인 필요.
   - [모름 ] 본문: Pretendard 폰트
   - [모름 ] 제목/강조: Noto Serif KR 폰트

---

### 📝 회원가입/로그인 (`/auth/signup`, `/auth/login`)

1. **회원가입**
   - [비정상 ] 회원가입 폼 UI가 정상인가? : 닉네임은 왜 안 받음? 이름과 닉네임 둘다 필요한데?
   - [됨 ] 이메일/비밀번호 입력 후 가입 가능?
   - [됨 ] 카카오 로그인 버튼 표시?

2. **로그인**
   - [됨 ] `member@test.com` / `test1234`로 로그인 가능?
   - [됨 ] 로그인 후 헤더에 사용자 정보 표시?
   - [됨 ] 로그아웃 버튼 동작?

---

### 📅 모임 상세 (`/meetings/[id]`)

모임 목록에서 아무 모임 클릭

1. **기본 정보**
   - [됨, 하지만 콩 아이콘이 갈색인데 초록색 계열이었으면 함 ] 모임 제목, 일시, 장소, 참가비 표시?
   - [됨 ] 환불 규정 섹션 표시 ("더보기"로 접힘)?

2. **신청 버튼**
   - 비로그인: [됨 ] 로그인 유도 표시?
   - 로그인 후: 
     - [됨 ] "신청하기" 버튼 클릭 시 결제 모달 표시?
     - [됨 ] 간편결제/계좌이체 탭 전환 동작?

---

### 💳 결제 플로우 (로그인 필요)

1. **간편결제 탭**
   - [바로 카카오 페이로 넘어감. 토스페이라는 글자를 빼야할 듯] 결제 수단 선택 화면 표시?
   - [우리 토스페이는 아직 안쓰는걸로 앎 ] 카카오페이/토스페이 등 아이콘 표시?
   - [됨 ] 결제 버튼 클릭 시 PortOne 팝업 호출? (테스트 모드)

2. **계좌이체 탭**
   - [됨 ] 은행, 계좌번호, 예금주 정보 표시?
   - [됨 ] 입금자명 자동 생성 (MMDD_이름 형식)?
   - [됨 ] 계좌번호 복사 버튼 동작?
   - [됨 ] "입금했습니다" 체크 후 완료 버튼 활성화?

---

### 👤 마이페이지 (`/mypage`) - 로그인 필요

1. **통계 영역**
   - [0이라 되어있음 ] "총 N회 참여" 표시?
   - [됨 ] "지독해와 함께한 지 N일째" 표시?
   - [안보임 ] 획득 배지 목록 표시?

2. **신청 내역**
   - [입금대기일 땐 취소가 안됨. ] 확정/대기/취소 구분 표시?
   - [확인불가 ] 취소 버튼 클릭 시 모달 표시?

3. **취소 모달** : 대기 1번째 모임 선택 시 마감되었씁니다 버튼만 있고 취소 불가
   - [ ] "마음 돌리기" 화면 표시?
   - [ ] 함께 신청한 회원 표시?
   - [ ] 환불 금액 자동 계산?
   - [ ] 취소 사유 선택 필수?

### 🎉 칭찬하기 & 배지 (참여 완료된 모임 필요) :이건 어떻게 하라고 나보고?

   - [ ] 같이 참여한 회원 목록 표시?

   - [ ] 5개 칭찬 문구 선택 가능?
   - [ ] 칭찬 완료 시 완료 페이지 이동?

2. **배지 획득**
   - [ ] 배지 획득 시 Confetti 효과?
   - [ ] 배지 모달 표시?
   - [ ] 마이페이지에서 배지 목록 확인?

### ⚙️ 운영자 페이지 (`/admin`) - 운영자 계정 필요

`admin@test.com` 또는 `super@test.com`으로 로그인

1. **대시보드**
   - [ ] 통계 카드 표시 (참가 현황, 수입/환불, 재참여율)? 보이긴 함.
   - [ ] 월 필터 동작? :하는 것 같음

2. **입금 관리** (`/admin/transfers`)
   - [ ] 입금대기 목록 표시? :됨
   - [ ] 모임별 필터 동작? : 안보임 안되는 듯. 모임리스트가 뜨긴 뜸.
   - [ ] 입금 확인 버튼 동작? : 동작함

3. **권한 관리** (`/admin/permissions`) - super_admin만
   - [ ] super_admin 계정에만 메뉴 표시? : 맞음
   - [ ] 운영자 권한 체크박스 토글 동작? : 동작

4. **배너 관리** (`/admin/banners`)
   - [ ] 배너 목록/추가/수정/삭제? : 수정은 URL넣어야하는데 나중엔 이미지 추가기능 필요할 듯 추가 수정 삭제 됨. 
   - [ ] 드래그 앤 드롭 순서 변경? : 가능
