## 사전 준비

### 1. 테스트 계정 생성 (Supabase 대시보드)

1. Supabase 대시보드 접속
2. **Authentication > Users > Add user**
3. 아래 3개 계정 생성:

| 이메일 | 비밀번호 | 역할 | 용도 |
|--------|----------|------|------|
| `super@test.com` | `test1234` | super_admin | 최고 관리자 |
| `admin@test.com` | `test1234` | admin | 일반 관리자 |
| `member@test.com` | `test1234` | member | 일반 회원 |

### 2. 시드 데이터 삽입 (Supabase SQL Editor)

1. `jidokhae/supabase/test-seed-data.sql` 내용 실행
2. 실행 전 계정 생성이 완료되어야 함

### 3. 로컬 서버 확인

- 서버 주소: http://localhost:3001
- 서버가 꺼져있으면: `cd jidokhae && npm run dev`
