Claude Code의 단축키와 명령어를 정리하려 준비했습니다.

Claude Code의 주요 슬래시 커맨드(/)와 키보드 단축키를 정리해 드릴게요.

기본 슬래시 커맨드
커맨드	설명
/help	사용 가능한 모든 커맨드 표시
/exit	Claude Code 종료
/clear	대화 기록 초기화
/compact [instructions]	대화 내용을 요약 (선택적 지시사항 포함)
/config	설정 패널 열기
/doctor	Claude Code 설치 상태 점검
/cost	현재 세션의 토큰 사용량 및 비용 표시
/login	계정 로그인/전환
/resume	이전 대화 이어서 진행
/ide	IDE 통합 관리
/mcp	MCP 서버 기능 접근
/hooks	후크 설정 (인터랙티브 메뉴)
/add-dir	추가 작업 디렉토리 지정
/bug	버그 신고 (대화 내용을 Anthropic에 전송)
/install-github-app	GitHub 앱 설치 (PR 자동 리뷰 기능)
키보드 단축키
단축키	설명
Ctrl + C	현재 작업 취소
Ctrl + D	Claude Code 종료
Ctrl + L	터미널 화면 지우기
Tab	자동 완성
↑ / ↓	명령어 히스토리 탐색
Esc (2번)	이전 메시지로 돌아가기
특수 입력
입력	설명
# (줄 시작)	CLAUDE.md에 메모리 추가
/ (줄 시작)	슬래시 커맨드 실행
! (줄 시작)	쉘 명령어 직접 실행 (예: !ls -la)
@파일명	파일/디렉토리 참조
CLI 명령어 옵션
bash
claude                      # 인터랙티브 REPL 시작
claude "질문"               # 초기 프롬프트와 함께 시작
claude -c                   # 가장 최근 대화 이어서
claude -p "질문"            # Headless 모드 (비대화형)
claude -r "세션ID" "질문"   # 특정 세션 재개
claude --version            # 버전 확인
claude update               # 최신 버전으로 업데이트
claude --debug              # 디버그 모드
