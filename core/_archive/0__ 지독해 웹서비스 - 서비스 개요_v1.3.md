## 1. 서비스명
**지독해 (가칭: 지독해 웹)**

---

## 2. 서비스 한 줄 정의
> 지독해 회원들이 모임과 쉽게 연결되어 있다는 느낌을 유지할 수 있게 해주는 독서모임 운영 플랫폼

---

## 3. 해결하려는 문제

**회원 관점**
- 카카오톡 일정이 한눈에 안 들어와서 모임 확인이 불편함
- 참가 신청(입금자명 수정 → 계좌이체 → 댓글)이 번거로움
- 취소/환불 요청 시 개인 연락 → 확인 → 처리까지 시간이 오래 걸림
- 바쁜 일상에서 신청해놓고 까먹거나, 뭘 신청했는지 헷갈림
- 리마인드가 없으면 모임 존재 자체를 잊어버림

**운영자 관점**
- 환불 요청 시 입금 내역을 일일이 찾아서 확인해야 함
- 이번 달 수입, 공간대여비, 재참여율 등 현황 파악이 어려움
- 정보가 카톡, 구글시트, 은행앱 등에 흩어져 있어 기억에 의존해야 함

---

## 4. 본질

**표면:** 복잡한 프로세스를 간소화하고 싶다

**본질:** 회원들의 부담을 줄여주고, 지독해에 소속감과 자부심을 느끼게 해주고 싶다

> "바쁜 현생을 사는 회원들이 부담 없이 자연스럽게 참여하고, 자기가 얼마나 꾸준히 함께했는지 느낄 수 있게"

---

## 5. 핵심 가치

| 가치 | 설명 |
|------|------|
| **쉬운 연결** | 일정 확인부터 신청/결제/취소까지 최소한의 동작으로 |
| **잊지 않게** | 자동 리마인드로 모임과의 연결 유지 |
| **소속감** | 활동 기록, 배지, 칭찬, 책장을 통해 "나는 지독해 회원"이라는 느낌 |
| **신뢰** | 결제/환불이 투명하고 빠르게 처리됨 |

---

## 6. 타겟 사용자

**주 사용자: 지독해 기존 회원**
- 정기모임에 참여하거나 참여하려는 회원
- 바쁜 일상 속에서 독서모임을 유지하고 싶은 사람

**부 사용자: 신규 회원**
- 지독해를 처음 알게 된 사람
- 후킹 랜딩페이지를 통해 유입되어 첫 정기모임을 신청하는 사람

**운영자: 영탁님(단무지)**
- 모임 운영, 정산, 회원 관리를 담당
- 다른 운영진에게 일부 권한 위임 가능

---

## 7. 핵심 규칙 (서비스에 반영)

- 모든 모임 참여의 전제조건 = 정기모임 참여 이력
- 6개월 이내 정기모임 참여 기록 없으면 다른 모임 참여 불가
- 환불 규정:
  - 정기모임: 3일 전 100% / 2일 전 50% / 이후 불가
  - 토론모임: 2주 전 100% / 7일 전 50% / 이후 불가
- 참가비 단위: "콩" (예: 10,000콩)

## 8. 주요 기능

### MVP (1차 개발)

**일정 경험**
- 비주얼 캘린더 (마감/대기가능/마감임박 상태 표시)
- 컴팩트 카드형 일정 UI (한 화면에 여러 일정, 참여하고 싶은 디자인)
- 이번 주 모임 하이라이트
- 참가 인원 표시 (정원은 숨김, 비교되지 않도록)
- 참가자 목록 (일부만 공개, 칭찬 많이 받은 회원 우선 표시)

**신청/결제**
- 간편결제 (카카오페이, 토스페이먼츠 등)
- 대기 신청 (자리 나면 알림 → 그때 결제)

**취소/환불**
- 셀프 취소 + 자동 환불 (규정에 따라)
- 취소 사유 입력 필수 (운영 개선용)
- 대기자 자동 승격 (취소 발생 → 순차 알림)

**리마인드**
- 자동 리마인드 알림톡 (모임 3일 전, 1일 전, 당일)
- 매월 말 정기 리마인드 (신청 모임 없는 회원에게도)
- 알림 템플릿 관리 (운영자가 문구/발송 시점 수정 가능)

**소속감/자부심**
- 마이페이지 활동 기록 (총 참여 횟수, 연속 참여, "함께한 지 OOO일째")
- 배지 시스템 (첫 참여, 10회, 연속 4주 등)
- 모임 종료 3일 후 칭찬하기 + 건의하기 + 후기 남기기 기능
- 정기모임 참여 자격 표시 ("마지막 참여: O월 O일", 6개월 임박 시 리마인드)
- 내 책장 (함께 읽은 책 기록, 한 문장 기록)

**신규 회원**
- 후킹 랜딩페이지 (브랜드 스토리, 분위기, 회원 후기)
- 일정 클릭 시 안내 팝업: "지독해가 어떤 곳인지 먼저 보시는 걸 추천해요."
- 첫 모임 전날 환영 알림 (준비물, 장소 안내)
- 첫 모임 후 후기 요청 + 다음 모임 안내

**운영자 기능**
- 대시보드 (이번 달 참가 현황, 수입, 환불 내역, 재참여율)
- 요청함 (환불 요청, 문의 확인 + 답변 기능)
- 홈 화면 콘텐츠 배너 관리
- 알림톡 수동 발송 (공지, 이벤트 안내 등)
- 운영자 권한 관리 (다른 운영진에게 권한 선택적 부여)

---

### 추후 기능 (2차 이후)

- 결제 수단 미리 등록 (원클릭 결제)
- 대기 신청 시 선결제 → 자동 환불
- 구글/애플 캘린더 자동 추가
- 정기모임 개설 요청 (회원들이 원하는 지역/요일 투표)
- 다른 회원 책장 구경
- 책 등록 배지 (10권, 50권 등)
- 참여 리포트 (월말/연말 "올해 OO회 참여")
- 클럽장 콘텐츠 정산 자동화 (80/20 자동 계산)
- 토론모임, 지글챌, 번개모임 전용 기능

## 9. 사용자 경험 흐름

**기존 회원**
```
웹 접속 
→ 홈: 콘텐츠/홍보 배너 + 이번 달 일정(카드형) + 내 신청 모임 D-day
→ 일정 클릭 → 상세 보기 or 바로 신청
→ 결제 (간편결제)
→ 완료 → 모임 화면 복귀
→ (모임 3일 전, 1일 전, 당일) 리마인드 알림
→ 모임 참여
→ (3일 후) 칭찬하기/참여했어요/후기 남기기 알림
```

**신규 회원**
```
웹 접속 → 일정 화면
→ 일정 클릭 → 안내 팝업:
   "지독해가 어떤 곳인지 먼저 보시는 걸 추천해요."
   
   [후킹페이지 보러가기]  [바로 신청하기]

→ (후킹페이지 선택 시) 브랜드 스토리, 분위기 확인 → 신청
→ (바로 신청 선택 시) 결제 진행
→ 첫 모임 전날 환영 알림 (준비물, 장소 안내)
→ 첫 모임 참여
→ (3일 후) 후기 요청 + 다음 모임 안내
```

**취소/환불**
```
마이페이지 → 신청 내역 → 취소하기
→ 마음 돌리기 화면 (함께 신청한 회원, 모임 임박 등 표시)
→ 취소 사유 선택 (필수)
→ 환불 규정 안내 + 동의
→ 최종 확인 후 자동 환불
→ 대기자 있으면 자동 승격 알림
```
※ 취소는 신청보다 마찰을 높여 충동적 취소 방지 (상세: PRD 섹션 6)

## 10. 기대 효과

**회원**
- 모임 확인/신청/취소가 쉬워져 참여 허들이 낮아짐
- 리마인드로 모임을 잊지 않고, 지독해와 연결된 느낌 유지
- 활동 기록과 책장을 통해 소속감과 자부심

**운영자**
- 정산/환불 처리 시간 대폭 단축
- 모든 정보가 한 곳에 → 기억에 의존하지 않아도 됨
- 재참여율 등 데이터 기반 운영 가능
- 알림 문구/시점을 유연하게 조정 가능

**지독해 전체**
- 회원 이탈 감소, 재참여율 증가
- 신규 회원 유입 경로 확보 (후킹 랜딩페이지)
- 브랜드 이미지 강화 ("체계적이고 신뢰할 수 있는 모임")

## 11. 브랜드 톤

따뜻하고 편안하면서도, 고급스럽고 진지하게 느껴지는 분위기. 가볍게 소비되는 모임이 아니라, 가치 있는 시간을 함께하는 커뮤니티라는 인상.
